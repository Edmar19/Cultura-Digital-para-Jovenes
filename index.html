Objetivo del Proyecto

Ofrecemos un sitio web educativo gratuito y acciones formativas para jóvenes de nivel medio superior, con el fin de:

  • Fomentar la seguridad digital.
  • Potenciar la productividad con herramientas tecnológicas.
  • Brindar acceso a recursos educativos de calidad.
